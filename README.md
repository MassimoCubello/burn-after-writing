# burn-after-writing

Burn After Writing is a Useless Web application for Workshops in Web Development 2.

This application provides users with a private space to write emails they know they would never send. Users can write an email freely and then choose to “burn” it instead of sending it. Once released, the content is permanently removed from the interface.

The application does not send, store, or transmit any of the messages that are inputted into the text box. Its purpose is to provide a safe space for an emotional release while reinforcing the idea of letting go.
